## Description
This repository contains Java Object-Oriented Programming exercises. Key concepts include:
- Classes & Objects
- Inheritance
- Polymorphism
- Abstraction & Encapsulation

## Objectives
- Understand OOP principles in Java
- Create classes and objects
- Implement inheritance and polymorphism

## Contents
- `src/` : Java source code files
- `README.md` : this file
- `examples/` : OOP examples

## How to Run
1. Open IDE (Eclipse, IntelliJ)
2. Compile and run Java files from `src/` folder
